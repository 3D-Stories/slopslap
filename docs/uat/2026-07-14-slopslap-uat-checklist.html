Pass rule. UAT passes only if every P0 scenario passes. A P0 failure blocks release. Run each scenario the way a real user would — the slash commands /slopslap:audit|suggest|review|apply|feedback are the primary path; the deterministic CLIs are the reproducible power-user lane. CLI commands only run as python3 scripts/... from the repo root (cd /home/rocky00717/rawgentic/projects/slopslap first) — the scripts are not on PATH and not executable, so a bare feedback.py / assemble.py fails with "command not found".
