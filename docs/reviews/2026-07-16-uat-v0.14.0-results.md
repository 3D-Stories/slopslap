## Findings

1. **Genre-classifier weakness → #98 (real).** Both `marketing-heavy` briefings classify as `spec`
   (the enum has no `marketing`, and their structure trips the spec heuristic). `spec` keeps the
   cadence class, so their corporate slop recommends **keep** instead of **strip** — the tool is
   blunted on the "heavy" genre. Recommendation-quality gap only; the `spec` fallback is the safe
   (never-over-strip) direction. Filed as issue #98.
2. **`audit_status: flagged` is universal (by design, cosmetic).** Every doc — clean ones included —
   reads `flagged`, because the deterministic status means "measurable candidates exist," not
   "harm." Harm is the model lane's judgment. A naive reader could misread the status; worth a doc
   clarification, not a code change.
